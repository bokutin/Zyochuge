h2. 実行環境

OS付属のPerlを使う場合はそれを、自前で用意する場合はperlbrewやplenvが使えます。

必要なモジュールは cpanm コマンドを使います。
入っていない場合は下を参考に入れてください。
http://search.cpan.org/perldoc?App::cpanminus#INSTALLATION
